What's Cookin?

View Site View Source

Description: A simple wrapper for the Recipe Puppy API. Users can search for either recipe titles or ingredients, and the app returns ten recipes (see "What's Next?" below).

Technologies Used: HTML, CSS, Grid, JavaScript, Node.js, Express.js, EJS

Challenges Overcome: The goal of this project was to call on an API and present the supplied data in a meaningful way. I utelized Express and Request for the back-end routing and EJS to display data from the API dynamically within the HTML. CSS Grid (specifically it's grid-template-columns: minmax feature) allowed the recipes to display responsively and play nice on mobile.

What's Next? One of the biggest things missing from this app is pagination! The Recipe Puppy API is set up in such a way that it automatically returns only ten results for each search query. To see more results, you have to append "p=2", 3, 4, etc. to the endpoint url. I haven't yet figured out how to gracefully code that into the UI, but I'm working on it!

Fahrenheit to Celsius Converter

View Site View Source

Description: A simple web app that will convert degrees Fahrenheit to degrees Celsius, and vice versa.

Technologies Used: HTML, JavaScript, jQuery, Bootstrap

Challenges Overcome: My goal with this project was to become more familiar with Bootstrap for CSS styling and jQuery for DOM manipulation. The logic for how to convert and display the temperature was quite simple, it was switching between the "Fahrenheit to Celsius" and "Celsius to Fahrenheit" states that took more thought and planning.

What's Next? I wanted to utelize Bootstrap as much as possible for this app (in fact, I didn't even include my own stylesheet), but I know there is a lot more room to be creative with the design and styling. My next step for this project is to customize the look and feel of it. There is especially a lot of room to draw attention to the switch between conversion states, since I consider this to be the most interesting feature of the app.

Personal Developer Portfolio

View Site View Source

Description: This site you're on right now! A simple single-page static website to showcase myself as a developer.

Technologies Used: HTML, CSS, JavaScript

Challenges Overcome: Apart from the design, the trickiest part about making this site was coding the pop-up modal. In order to keep the code light and scalable, I chose to use the getElementsByClassName selector to return node lists of all of the buttons and modals. Then I ran through them all with a for-loop and put the event listeners inside. This way I don't have to write a brand new block of JavaScript for each new project I add; once I add it to the HTML, the pop-up modal will automatically work.

What's Next? Apart from the seemingly neverending struggle to find the right fonts, my biggest issues so far with the site are compatability between different browsers. Smooth-scrolling only works on Chrome right now, and my contact links at the bottom of the page are not formating correctly in Safari (probably pertaining to flexbox).
